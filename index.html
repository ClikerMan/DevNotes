Table tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The set of rows defining the footer in a table.
Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

Colors in CSS

Color type Syntax Description
Color name red Simply type the name of the color.
HEX #23F41A The set of rows defining the column headers in a table.
RGB rgb(100, 255, 23) RGB colors range from 0 to 255, with one value for red, blue and green
HSL hsl(254, 50%, 50%) HSL stands for hue (the color itself), saturation (the intensity of the color), and lightness (how light or dark a color is). Hue ranges from 0 to 360 and saturation and lightness are both represented as percentages
Opacity / Alpha
RGBA rgba(24, 31, 21, 0.5) Alpha (opacity) - works only for RGB, HEX and HSL. It adds opacity to colors, it ranges from 0-1 (0.2 or 0.5 or 0.8) with examples underlined in the syntax row. For HEX, you apply opacity by adding 2 more digits at the end of the color string like so: #frfrfr50
HSLA hsla(243, 50%, 50%, 0.7)
